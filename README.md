# 入出荷台帳

プゥル・ヴー向けの入出荷台帳アプリ（タブレット入力）。

**正本:** `index.html`

## 伝票アプリについて

**プゥルヴー伝票は変更しません。** 今までどおり単独で使えます。

台帳アプリは伝票の客先・品目データを**読むだけ**です。伝票への数量は「伝票入力メモ」画面を見ながら、伝票アプリへ手入力します。

## 日常の使い方

1. 台帳アプリを開く（**パスワードのみ**・伝票と同じ）
2. 客先を選ぶ → 品目一覧が出る
3. 入/出をタップ → 右のテンキーで数字入力
4. **完了して保存**
5. 納品伝票が必要なとき → **伝票入力メモ** で「出」を確認し、**伝票アプリ**で発行
